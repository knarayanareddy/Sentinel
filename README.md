# Sentinel

**Enterprise Covenant Monitoring Agent** — RAISE Summit Hackathon 2026  
Vultr / VultronRetriever Track · Remote Solo Entry

> ⚠️ **Hackathon project in progress.** This repository is being built during the RAISE Summit Hackathon 2026 (July 4–5, 2026). Documentation and application code are being added in phases.

SENTINEL is a multi-step enterprise document agent powered by VultronRetriever models on Vultr Serverless Inference. It ingests financial documents, plans an investigation, retrieves from a vector store across multiple passes, reasons over evidence, calls tools, and — before executing any irreversible action — routes through a human oversight gate that produces a cryptographically-hashed, citation-backed incident record.

**Domain:** Finance — Loan Covenant Breach Monitoring
